# README

A simple, searchable api service for a hypothetical set of users.

Prerequisites:

* Ruby 2.3 >=

* Rails 5.0 >=

* JRE for solr

* `brew install redis`

Running the app:
*  update database.yml with credentials
*  copy .env.sample to .env and update
* `rails db:create`
* `rails db:migrate`

* `foreman start`
